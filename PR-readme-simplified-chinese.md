# PR: README.md 繁体中文转简体中文

## 摘要

将 `README.md` 全文从**繁体中文（zh-TW）**转换为**简体中文（zh-CN）**，统一项目文档语言标准，与前端 UI（T10 已完成）保持一致。

## 背景

T10 已将前端 19 个源文件（`.vue` / `.ts` / `.html`）通过 OpenCC t2s 统一转换为简体中文，并设置了 `lang="zh-CN"`。但 `README.md` 作为项目门面文档，仍全部使用繁体中文（如「專案」「資料庫」「欄位」「資訊」等）。

## 改动内容
